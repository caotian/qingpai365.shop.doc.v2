## 请求绑定数据

```
POST /app/customer/wechat-bind/qrcode
```

> 请求数据: shopId: 门店ID customerId(新顾客不传): 顾客ID


## 二维码

```
GET /api/qrcode
```

> 请求数据: content: 二维码内容

## 请求状态

```
GET /app/customer/wechat-bind/status
```

> 请求数据: bindKey: 绑定Key
