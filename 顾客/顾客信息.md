## 列表

```
GET /app/customer/customer
```

## 添加

```
POST /app/customer/customer
```

## 更新

```
PATCH /app/customer/customer/:id
```

## 删除

```
DELETE /app/customer/customer/:ids
```

## 添加画像

```
POST /app/customer/customer/tag-add
```

> 请求: customerId: 顾客ID tag: 标签 

## 删除画像

```
POST /app/customer/customer/tag-remove
```

> 请求: customerId: 顾客ID tag: 标签
