## 上传Excel

```
POST /pc/product/upload/excel
```

> 请求参数: {"category":"分类","upfile":"上传的excel文件, 支持xls和xlsx"}

>返回数据: 原始excel内容, 直接显示即可

## 批量添加

```
POST /pc/product/product/batch-insert
```

> 请求参数: {"category":"分类","dataList":"上传接口返回的内容(data参数)"}
